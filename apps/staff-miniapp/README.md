# Zero便利超市 员工端小程序

微信小程序 AppID：`wx1d0c7b230e8730dc`

## 当前范围

- 工作台：待接单、待出库、配送中、最近入库和历史查询入口
- 商品入库：扫码识别商品，选择入库门店/货架，填写数量并生成入库记录
- 订单处理：按状态筛选，接单、扫码出库、异常上报、进入配送
- 拣货出库：逐项扫码核验、异常上报、完成出库并同步订单状态
- 配送任务：地图预览、开始配送、联系用户、确认送达
- 历史配送：按订单号、用户、地址和时间范围查询已完成配送订单

## 打开方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择 `apps/staff-miniapp`。
4. AppID 使用 `wx1d0c7b230e8730dc`。
5. 当前为演示版，接口和库存均使用本地模拟数据。

## 本地 API 联调

先在仓库根目录启动 API：

```bash
node services/api/server.js
```

小程序会请求：

```text
http://127.0.0.1:8787
```

微信开发者工具中需要勾选“不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书”。
