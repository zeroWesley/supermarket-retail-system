# Zero便利超市 员工端小程序

微信小程序 AppID：`wx1d0c7b230e8730dc`

## 当前范围

- 工作台：待接单、待拣货、配送中、异常提醒
- 订单处理：接单、查看地址、进入拣货
- 拣货核验：逐项核验、异常上报、完成拣货
- 扫码出入库：手机扫码模拟、入库/出库切换、库存记录
- 配送任务：开始配送、确认送达、联系用户

## 打开方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择 `apps/staff-miniapp`。
4. AppID 使用 `wx1d0c7b230e8730dc`。
5. 当前为演示版，接口和库存均使用本地模拟数据。

## 本地 API 联调

先在仓库根目录启动 API：

```bash
node services/api/server.js
```

小程序会请求：

```text
http://127.0.0.1:8787
```

微信开发者工具中需要勾选“不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书”。
