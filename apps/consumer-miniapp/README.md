# Zero便利超市 C端用户小程序

微信小程序 AppID：`wx96dae1169f62c178`

## 当前范围

- 首页：门店、搜索、优惠券、分类入口、场景推荐、商品推荐
- 分类：左侧分类、商品列表、加购
- 商品详情：价格、库存、活动、配送承诺、加入购物车
- 购物车：数量调整、优惠提示、模拟支付提交订单
- 订单：订单状态、履约时间线、售后入口
- 我的：优惠券、订单、客服帮助、地址与常用服务

## 打开方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择 `apps/consumer-miniapp`。
4. AppID 使用 `wx96dae1169f62c178`。
5. 当前为演示版，接口和支付均使用本地模拟数据。

## 本地 API 联调

先在仓库根目录启动 API：

```bash
node services/api/server.js
```

小程序会请求：

```text
http://127.0.0.1:8787
```

微信开发者工具中需要勾选“不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书”。
