# Zero便利超市网页后台

这是管理端和运营端的第一版静态演示应用，无需安装依赖，可直接打开 `index.html`。

## 当前范围

运营端：

- 经营概览
- 商品管理
- 库存管理
- 营销活动
- 订单管理
- 门店配置

管理端：

- 账号管理
- 角色权限
- 操作日志

## 演示账号

管理员：

- 账号：`admin`
- 密码：`admin123`
- 权限：可选择进入管理端或运营端，默认拥有全部后台菜单

普通运营人员：

- 账号：`operator`
- 密码：`ops123`
- 权限：只能进入运营端，默认仅授予“商品管理”菜单

## 权限规则

- 管理端仅系统管理员可进入。
- 系统管理员默认拥有全部后台权限，不需要逐项授予。
- 运营人员按账号单独授予运营端菜单权限，例如只授予“商品管理”，不授予经营概览、库存管理、营销活动或订单管理。

## 数据说明

当前数据存储在浏览器 `localStorage` 中，支持新增商品、新建活动、库存修正、订单状态流转、账号启停等演示操作。

后续接入后端 API 时，将把 `app.js` 中的本地数据读写替换为接口请求。
